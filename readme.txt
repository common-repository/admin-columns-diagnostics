This plugin assists you with support for Admin Columns.

== Description ==

It shows a tab with diagnostic info about your site and settings. You can use this plugin when you have a support issue.

== Installation ==

1. Install and activate the plugin
2. Go to Settings > Admin Columns
3. Click on the Debug tab to open the debug page
4. Click on the 'Copy to clipboard' button or select the text and copy the content and paste it into an email to our support

= Related Links =

* [Admin Columns](https://wordpress.org/plugins/codepress-admin-columns/)

== Frequently Asked Questions ==

= Do I need Admin Columns to be installed? =

Yes, in order to make this plugin work, you need Admin Columns to be installed on your environment.
